pub struct SlotTimestampProvider { /* private fields */ }
Expand description

Provide duration since unix epoch in millisecond for timestamp inherent. Mocks the timestamp inherent to always produce a valid timestamp for the next slot.

This works by either fetching the slot_number from the most recent header and dividing that value by slot_duration in order to fork chains that expect this inherent.

It produces timestamp inherents that are increased by slot_duration whenever provide_inherent_data is called.
